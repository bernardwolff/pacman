Pacman Help

Play Keys

  • arrow keys - move pacman
  • space - pause
  • f - while paused, move to next frame of animation

Level editing

  • Click on any pill or wall to select it.
  • Once you have selected an object, use the arrow keys to move it around the screen.
  • r - rotate object
  • d - delete object
  • g - grow object
  • s - shrink object
  • n - select next object
  • p - select previous object
  • u - unselect selected object. Can also click on the background.
  • a - add line
  • i - add pill
  • l - log all lines to javascript console
